Projeto Linux + AWS

Configuração de Servidor Web com Monitoramento

Sobre o Projeto

Este projeto visa configurar uma infraestrutura robusta utilizando Linux e AWS, abrangendo VPC, EC2, Nginx, monitoramento e automação.

Principais Tecnologias

  • Amazon VPC
  • Instâncias EC2
  • Nginx como Proxy Reverso
  • Monitoramento e Observabilidade
  • Automação com scripts e ferramentas DevOps

Instalação do Nginx dentro da Instancia AWS


        # Instalando o nginx no Ubuntu virtual
        sudo apt-get install

        # Inicio o sistema nginx
        sudo systemctl start nginx

        # Ativo o sistema nginx
        sudo systemctl enable nginx

Alteração Página Web


        # Acesso o diretório onde encontro o html
        cd var/www/html 

        # Crio a página 'index.html' com meu site
        sudo nano index.html

        # Crio o arquivo CSS para estilização 'styles.css' 
        sudo nano styles.css

Código bash para monitoramento do sistema:


            #!/bin/bash

            while true; do
                if ! systemctl is-active --quiet nginx; then
                    echo "⚠️ Nginx caiu! Reiniciando e enviando notificação..."
                    echo "$(date '+%Y-%m-%d %H:%M:%S') - O site caiu e foi reiniciado" >> /var/log/nginx_monitoramento.log
                    python3 /mnt/h/ACADEMIC/DEV_COMPASS_UOL/1-PROJETO-Linux+AWS/BACK_END/telegram_notify.py
                fi
                sleep 10  # Verifica a cada 10s
            done
